Premise

It is possible to access the Client API and Web API from the debug console in a browser. This allows you to write (and more importantly test) Client API calls and Web API calls without having to update and publish web resources each time.

Form Context

The equivalent of the Client API form context can be accessed as so:

Xrm.Page.ui._formContext

This then enables access to the typical form context functionality, such as accessing form controls and attributes.

Web API

The equivalent of the Dynamics Web API can be acessed as so:

Xrm.WebApi.online

The Web API can then be utilised as normal:

Xrm.WebApi.online.retrieveRecord("contact", contactGuid).then(
    (success) => {
        console.log(success);
    },
    (error) => {
        console.log(error);
    }
);
