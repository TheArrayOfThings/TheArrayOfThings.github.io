The Map

This webpage has a version of the wild camping map of dartmoor, taken from government data.

What went into this

This project involved the following:

1): I downloaded the government provided shapefile from here

2): I converted this shapefile to a geoJSON file using mapshaper.org

3): I ran the geoJSON through geojsonlint.com and observed multiple Polygons and MultiPolygons should follow the right-hand rule errors

4): I used mapster's Right Hand Rule GeoJSON Fixer and downloaded the fixed geoJSON

5): Finally, I used google maps Javascript API to display a map with the geoJSON overlayed - which is what you see here :)
